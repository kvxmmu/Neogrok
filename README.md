# Neogrok [WIP]

Neogrok is a free and open source self-hosted alternative to ngrok.io

# Features

- [x] Forward packet compression/decompression
  - [x] deflate
  - [x] zstd
- [x] TCP port forwarding
- [ ] UDP port forwarding
- [ ] Database interactions
- [ ] HTTP REST & web dashboard
- [x] Well done client
  - [ ] GUI
  - [x] CLI
